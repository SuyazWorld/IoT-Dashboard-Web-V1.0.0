# IoT-Dashboard-Web-V1.0.0

<h1 align="center">V1.0.0</h1> 

 <p style="text-align: justify; text-justify: inter-word;">This is an IoT dashboard web that is still intended for experimentation. This dashboard is used to receive data from ESP32 devices, several sensors and actuators. In addition to receiving data, this dashboard can also be used to send data and also control several actuators such as LEDs, servos, relays, and many more. In addition, this web can also store incoming data and display it in the form of tables and graphs.</p>  
  
<p>This Iot Dashboard Website using HTML,CSS,javascript for the Front-End, MQTT.js for the mqtt, PHP for the Back-end, and also </p>
